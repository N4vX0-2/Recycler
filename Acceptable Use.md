# Acceptable Use Notice

This project is intended only for lawful, authorized security work.

The purpose of the system is to help security practitioners keep client engagement artifacts isolated from persistent tooling and operator configuration. It is designed for cleanup, repeatability, and engagement hygiene. It is not designed to enable unauthorized access, conceal malicious activity, destroy evidence of wrongdoing, or evade detection.

## Allowed Use

Allowed use includes:

- authorized penetration testing;
- vulnerability assessment;
- defensive security testing;
- internal lab work;
- training environments;
- authorized red-team or purple-team exercises;
- system administration and cleanup of lawful testing artifacts;
- research where the researcher has proper authorization.

## Prohibited Use

Prohibited use includes:

- unauthorized scanning, exploitation, access, or testing;
- credential theft or data theft;
- malware, ransomware, phishing, fraud, or extortion;
- evasion, concealment, or cleanup of unauthorized activity;
- harassment, privacy invasion, sabotage, or service disruption;
- misrepresenting the project or creator(s) as endorsing malicious conduct;
- commercial resale, bundling, or product integration without express written permission.
